# graphql_tools_server
This is a backend project built with Node.js, Express.js, and Graphql 

### Installation

Clone the app
* ```https://github.com/SamuelSabasNayo/graphql_tools_server.git```

Install all the packages
* ```npm install ```

Run the server
*  ```npm start:dev ```

## Testing
Run Test case
* ```npm run test```
